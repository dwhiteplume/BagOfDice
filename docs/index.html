Bag of Dice

by @dariusw@tech.lgbt

See repository for more info.

Bag of Dice is a small powershell module with a set of tools to allow for simulated dicing operations.

Functions

There are five (5) basic functions included:

Get-BoDCoinFlip

This function is used to get a 50/50 chance response. It will return values of three types:

Option Type Result Set
Coin string 'Heads', 'Tails'
Binary int32 0, 1
Boolean bool True, False
