Bag of Dice

by @dariusw@tech.lgbt

See repository for more info.

Bag of Dice is a small powershell module with a set of tools to allow for simulated dicing operations.

Functions

There are five (5) basic functions included:

Get-BoDCoinFlip

This function is used to get a 50/50 chance response. It will return values of three types:

Option Type Result Set
Coin string 'Heads', 'Tails'
Binary int32 0, 1
Boolean bool True, False

Get-BoDDiceRolls

This function allows you to specify how many dice of a single type to roll.

# Example

Get-BoDDiceRolls -Dice 3 -Side 6

# Rolls 3d6 (three six-sided dice)

# Results are like:

DiceRolls         : {5, 5, 3}
Count             : 3
Average           : 4.33333333333333
Sum               : 13
Maximum           : 5
Minimum           : 3
StandardDeviation : 1.15470053837925
Request           : Get-BoDDiceRolls -Dice 3 -Side 6
StandardNotation  : 3d6

The resulting object includes the dice rolled (generic list), sum of the rolls, the issued command (Request), standard dice notation, and statistics provided by the Measure-Object commandlet.
